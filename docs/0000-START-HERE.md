# O que é o MedFlow

O MedFlow é uma plataforma SaaS para gestão de clínicas e profissionais da saúde.

Não é apenas um aplicativo.

Não é apenas um sistema de consultas.

É uma plataforma completa preparada para crescer continuamente.
